# Task 1 - Spaceship Titanic Classification

## Overview

This project focuses on predicting whether passengers were transported to another dimension using the Spaceship Titanic dataset.

The task is a binary classification problem where the target variable is:

- `Transported` → Passenger transported (True/False)
